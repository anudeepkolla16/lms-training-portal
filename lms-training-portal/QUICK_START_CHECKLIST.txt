STEP 1: INSTALL DEPENDENCIES (5 minutes)
────────────────────────────────────────

[ ] Open PowerShell
[ ] Navigate to: C:\Users\AnudeepKolla\lms-training-portal
[ ] Run: npm install

What happens: Downloads React and all libraries needed


STEP 2: REGISTER APP IN AZURE AD (10 minutes)
──────────────────────────────────────────────

[ ] Go to: https://portal.azure.com
[ ] Search for "App registrations"
[ ] Click "New registration"
[ ] Name: "LMS Training Portal"
[ ] Select: "Accounts in this organizational directory only"
[ ] Click "Register"
[ ] SAVE these 2 values:
    - Application (client) ID → REACT_APP_CLIENT_ID
    - Directory (tenant) ID → REACT_APP_TENANT_ID
[ ] Click "Authentication" in sidebar
[ ] Click "+ Add a platform"
[ ] Choose "Single-page application"
[ ] Add redirect URIs:
    - http://localhost:3000
    - http://localhost:3000/
[ ] Click "Save"
[ ] Click "API permissions"
[ ] Click "+ Add a permission"
[ ] Search for "SharePoint" and click it
[ ] Select "Delegated permissions"
[ ] Check: AllSites.Read, Sites.Read.All, user.read
[ ] Click "Add permissions"


STEP 3: CREATE .ENV FILE (5 minutes)
────────────────────────────────────

[ ] In project folder, find ".env.example"
[ ] Copy it and rename to ".env" (without .example)
[ ] Open ".env" with Notepad
[ ] Fill in values from Step 2:
    REACT_APP_CLIENT_ID=paste_client_id_here
    REACT_APP_TENANT_ID=paste_tenant_id_here
    REACT_APP_AUTHORITY=https://login.microsoftonline.com/paste_tenant_id_here
    REACT_APP_SHAREPOINT_SITE=https://sarasanalytics.sharepoint.com/sites/training-library
[ ] Save file


STEP 4: TEST LOCALLY (10 minutes)
─────────────────────────────────

[ ] In PowerShell, run: npm start
[ ] Wait for "Compiled successfully!" message
[ ] Browser opens to: http://localhost:3000
[ ] Click "Sign In with Microsoft 365"

Test what you see:

[ ] Close modal and refresh - course status updated? ✓
